TECHNICAL SPECIFICATIONS
════════════════════════════════════════════════════════════════════════════

Video Output:
  • Resolution: 1080x1920 pixels (vertical/shorts format)
  • Codec: H.264 (universally compatible)
  • Bitrate: 5000 kbps video, 128 kbps audio
  • Format: MP4 container
  • Duration: Matches podcast audio length (~11 minutes)

Authentication:
  • Method: Google Service Account
  • Location: /home/aarevalo/.openclaw/agents/hermes/agent/google-service-account.json
  • Scope: YouTube Data API v3 (upload only)
  • Type: Direct authentication (no user prompts)

APIs & Tools:
  • YouTube Data API v3: Video upload and metadata
  • Google Auth Library: Service Account authentication
  • FFmpeg: Video encoding and compositing
  • FFprobe: Audio duration detection
  • Requests: HTTP client for Pexels API
  • XML Parser: RSS feed parsing (built-in)

Python Requirements:
  • google-auth==2.25.2
  • google-auth-oauthlib==1.2.0
  • google-auth-httplib2==0.2.0
  • google-api-python-client==1.12.5
  • requests==2.31.0

System Requirements:
  • Python 3.8+
  • FFmpeg with libx264 support
  • Unix/Linux environment
  • Cron daemon (for scheduling)
